2.1 Defination

定义分为两类：程序型定义（折叠）和筛选型定义（筛子）。

折叠就是对一连串的字符进行整体的代替，例如，\Delta \equiv b^2 - 4ac 和 [A, B] \equiv AB - BA。

筛子就是讲一个对象进行一定程序性操作后，能否实现规定的性质，例如，偶数和质数。

Def is the axioms of our game. These are the minunum steps we have. We start our exploration based on axioms.

下面是一系列的定义举例：

  • 复数：i^2 = -1
  • 组合：C^k_n = \dfrac{n!}{(n-k)!k!}
  • 阶乘：n! = 1\cdot 2\cdot 3 \cdots n
  • 求和：S_n = a_1 + a_2 + a_3 + \cdots + a_n
  • 对数：a^b =c \Rightarrow \log_a{c} = b
  • 离心率：e = \dfrac{c}{a}
  • 共轭复数：a+bi \Rightarrow a-bi
  • 双曲正弦：\sh{x} = \dfrac{e^x - e^{-x}}{2}
  • 傅里叶变换：F(\xi) = \dfrac{1}{\sqrt{2\pi}}\int^{\infty}_{-\infty}f(x)e^{i\xi x}dex
  • 向量的点积：\bold{a} \cdot \bold{b} = x_1x_2 + y_1y_2
  • 双曲线的渐近线：y = \pm\dfrac{b}{a}x
  • 三角形的周长：C = a+b+c

根据定义，可以命制的一系列题目：

  • 化简：(i+1)(i-1)
  • 求 C^2_5
  • 求 4!
  • 已知 a_n = 3n + 2 ，求 S_4
  • 求 \log_2{8}
  • 求 \dfrac{x^2}{4} + \dfrac{y^2}{3} =1 的焦点坐标
  • 已知 A = \{1,2,3,4 \}, B = \{2,3,4,5 \} ，求 A \cap B
  • 求 \dfrac{x^2}{4} - \dfrac{y^2}{3} =1 的渐近线方程
  • 已知 \bold{a} = (1,2), \bold{b} = (2,3)，求 \bold{a} \cdot \bold{b}
  • 求 \sh{(\ln{2})}
  • 已知回归方程 \widehat{y} = 3x -2 ，求样本点 (4,9)处的残差

对于程序性的定义，我们可以写出具体程序来表示这个定义，例如： 


      function eccentricity(ellipse) {
        return ellipse.c / ellipse.a
      }
    

      function residual(regression_equation, point) {
        return point.y - regression_equation(point.x);
      }
    

      function dot_product(vector_a, vector_b) {
        return vector_a.x * vector_b.x + vector_a.y * vector_b.y ;
      }
    

      function sum(array, n) {
        var sum = 0;
        for ( var i = 0; i < n; i++ ) {
          sum += array[i];
        }
        return sum;
      }

而根据程序，我们可以命制的习题，例如： 


      console.log(eccentricity(new Ellipse(2, 1)));
      console.log(residual(new Equation(3, -2), new Point(4, 9)));
      console.log(dot_product(new Vector(1, 2), new Vector(2, 1)));
      console.log(sum([1, 2, 3, 4, 5, 6, 7, 8], 3));

这样的习题，无论是命制还是解答都是非常简单的。对于一个解决机，它只要识别出是定义级别的操作，从而直接带入函数即可。对于一个解题者，TA只要记住程序的每一步，按部就班的执行即可。
