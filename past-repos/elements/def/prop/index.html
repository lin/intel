2.3 Properties and Formulas

性质和公式，一般是由定义推导出来的近距离的结论，其特点一般形式短小、使用频率高。

This is kind of shortcut we get while we explore the newly defined territory. So we don't need to revisit each time we need it purposely. We care it in several metrics: 1, frequency of usage.2,concise of the identities.3, short-range because of the frequency since we often visit the vicinity while we explore far.

Of those oceans of identities we can obtain, we prefer short identities and build from them, there'a easy to manipulate and remember.

  • a^{m+n} = a^ma^n
  • \sin{(A+B)} = \sin{A}\cos{B} + \cos{A}\sin{B}

  • C^k_n = C^{n-k}_n

  • y=x^2 是偶函数，其图像关于 y 轴对称
  • \sin{\dfrac{\pi}{12}} =\dfrac{\sqrt{6} -\sqrt{2}}{4}
  • (e^x)'=e^x
  • 等比数列中，S_n = \dfrac{a_1(1-q^n)}{1-q}
  • 余弦定理：a^2 = b^2+c^2 -2bc\cos{A}
  • 1^3+2^3+3^3+\cdots +n^3 = \dfrac{1}{4} n^2(n+1)^2
  • [AB, C] = A[B, C] + [A, C]B
  • -\ln{x} = \ln{\dfrac{1}{x}}
  • \sum\limits^n_{i=1} \dfrac{1}{(2i-1)(2i+1)(2i+3)} = \dfrac{1}{12}-\dfrac{1}{4(2n+1)(2n+3)}
  • a^3+b^3+c^3-3abc=(a+b+c)(a^2+b^2+c^2-ab-bc-ca)
  • \sin{x} < x < \tan{x}, (0 < x < \dfrac{\pi}{2})
  • \sin{x}>x-\dfrac{x^3}{6}, (x>0)
  • 对于正数，\dfrac{a_1+a_2+\cdots+a_n}{n} \ge \sqrt[n]{a_1 a_2\cdots a_n}
  • \tan{A} + \tan{B} + \tan{C} = \tan{A}\tan{B}\tan{C}
  • \sum\limits^n_{i=0} \dfrac{C^j_n}{j+1} = \dfrac{2^{n+1}-1}{n+1}
  • \sin{3\alpha} = -4\sin^3{\alpha}+3\sin{\alpha}
  • \ch{(x\pm y)} = \ch{x} \ch{y} \pm \sh{x}\sh{y}
  • S=\sqrt{p(p-a)(p-b)(p-c)}
  • 对于方程 x^3+px+q=0： x_1+x_2+x_3=0，\dfrac{1}{x_1}+\dfrac{1}{x_2}+\dfrac{1}{x_3} =-\dfrac{p}{q}，x_1 x_2 x_3=-q
  • 若矩阵可逆，则 \det A^{-1}= (\det A)^{-1}
  • \dfrac{d}{dx}f(\phi(x)) = f'(\phi(x))\phi'(x)
  • 椭圆切线方程：\dfrac{x_0 x}{a^2} + \dfrac{y_0 y}{b^2} =1
  • (\bold{a} \times \bold{b}) \cdot (\bold{c} \times \bold{d}) = (\bold{a} \cdot \bold{c}) (\bold{b} \cdot \bold{d}) - (\bold{a} \cdot \bold{d}) (\bold{b} \cdot \bold{c})
  • R_{jkl}i + R_{klj}i + R_{ljk}i = 0

对于程序性的定义，我们可以写出具体程序来表示这个定义，例如： 


      function eccentricity(ellipse) {
        return ellipse.c / ellipse.a
      }
    

      function residual(regression_equation, point) {
        return point.y - regression_equation(point.x);
      }
    

      function dot_product(vector_a, vector_b) {
        return vector_a.x * vector_b.x + vector_a.y * vector_b.y ;
      }
    

      function sum(array, n) {
        var sum = 0;
        for ( var i = 0; i < n; i++ ) {
          sum += array[i];
        }
        return sum;
      }

而根据程序，我们可以命制的习题，例如： 


      console.log(eccentricity(new Ellipse(2, 1)));
      console.log(residual(new Equation(3, -2), new Point(4, 9)));
      console.log(dot_product(new Vector(1, 2), new Vector(2, 1)));
      console.log(sum([1, 2, 3, 4, 5, 6, 7, 8], 3));

这样的习题，无论是命制还是解答都是非常简单的。对于一个解决机，它只要识别出是定义级别的操作，从而直接带入函数即可。对于一个解题者，TA只要记住程序的每一步，按部就班的执行即可。
